# 🔐 Password Generator

A modern Password Generator built using Python. This application generates strong and secure random passwords.

## 🚀 Features

- Generate random passwords
- Choose password length
- Includes uppercase & lowercase letters
- Includes numbers
- Includes special characters
- Simple and user-friendly interface

## 🛠️ Technologies Used

- Python
- CustomTkinter
- Random
- String
- Pyperclip

## 📂 Project Structure

```
Password-Generator/
│── password_generator.py
│── README.md
│── requirements.txt
```

## ▶️ Installation

Install the required libraries:

```bash
pip install -r requirements.txt
```

Or install manually:

```bash
pip install customtkinter pyperclip
```

## ▶️ Run the Project

```bash
python password_generator.py
```
